This map displays results of an ongoing breeding landbird monitoring program conducted by the National Park Service's Inventory and Monitoring Program. Each circle indicates the location of a permanent monitoring point where the bird community is sampled using standardized “point counts”. Hovering over and/or clicking on a circle will produce a popup with additional information. For detail on monitoring methods, select "Project Information" from the "About the Project" tab..

Base Map

You can select different base maps using the controls in the upper right.

  • Map: displays the NPS Park Tiles Standard OSM base map and is the default map with parks marked in green.
  • Imagery: displays the NPS Park Tiles satellite imagery with parks outlined in green.
  • Slate: displays the NPS Park Tiles Slate OSM base map which is similar to the default map but specifically tailored for low vision users.

Map Controls

Data to Map: This allows you to toggle between different types of data to display on the map. These include:
  • Individual Species: displays how many times a selected species was detected at each monitoring point. The value displayed can be filtered by sampling year, distance from observer, and visit (described below).
  • Number of Species: displays the number of species detected at each monitoring point. The value displayed can be filtered by sampling year, distance from observer, and visit (described below). Hovering over a point provides a popup that identifies the monitoring point and the total number of species detected. Clicking on a point will give you the full species list for that point.
  • Bird Community Index (BCI): displays the BCI score for each monitoring point. The BCI is an index of biotic integrity developed by O’Connell and others (1998, 2000) for central Appalachian bird communities. Higher BCI scores (indicating higher biotic integrity) describe a bird community in which specialists are well-represented relative to generalists. See "Project Information" on the "About the Project" tab for more information on this index.

Year: You can choose which year to display using the year selector.

Distance from Observer: When birds are detected during point counts, the observer estimates how far the bird is from the observer and assigns it to one of several distance bands. You can filter the results displayed by increasingly large distance bands using these options.

Visit: During each year of sampling, multiple point counts are conducted at each monitoring point (referred to as repeat visits). We do this because some birds are not detected by the observer conducting the point count and the repeat visits allow us to account for and correct for these “misses.” You can use these options to map the birds observed in each visit or the maximum number observed across all visits.

Species: This option is only available when "Individual Species" is selected under "Data to Map". This allows you to select which species to display.

Names: This option is only available when "Individual Species" is selected under "Data to Map". Use this to indicate if you want to use common names, Latin names, or American Ornithological Union (AOU) codes.

Zoom

Zoom to: You can select a park from the drop-down menu and click on "Go" to zoom the map to that park.

Point size: When a map is zoomed in, monitoring points are displayed with a default radius. The “point size” control allow you to choose among different radii to suit your visualization needs.
