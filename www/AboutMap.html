This map displays the results of an ongoing forest bird monitoring program conducted by the National Park Service in eleven parks in the Washington DC area. Each circle indicates the location of a bird monitoring point. Hovering over and/or clicking on a circle will produce a popup with additional infomration. For detail on monitoring methods, select "Project Information" from the "About" tab.

Map Controls

Data to map: This allows you to toggle between differentt types of data to display on the map. These include:
  • Individual Species: This is an esimtate of how many individuals of a given species were observed at each monitoring point during a given year.
  • Number of Species: This is the number of species observed at monitoring point during a given year. Clicking on a point will give you the full species list for that point.
  • Bird Community Index (BCI): This is an index that indicates if the bird community has birds typical of forest habitats ("High Integrity") or has more generalist birds ("Low Integrity"). See "Project Information" on the "About"" tab for more information on this index.

Distance from Observer: When birds are observed, the monitoring crew notes if the bird was 0-50 meters away, 50-100 meters away or further than 100 meters. Birds that are closer are easier to detect and identify, so the closer data may be more accurate. However, using all the data may gives a better indication of the species present in an area.

Species: This option is only avialable when "Individual Species" is chose under "Data to Map". This allows you to select which species to display.

Visit: This option is only avialable when "Individual Species" is chose under "Data to Map". This allows you map the birds observed in the first visit (usually late spring to early summer), the second visit (usaully early-mid summer) or the maximum number observed for either visit. As the same individual birds can be observed in both visits, it is recommened that you do not add the number of birds from the two visits.

Year: Monitoring points are visited twice each year. You can choose which year to display using the slider.

Names: Use this to indicate if you want to use common names, Latin names, or American Ornithological Union (AOU) codes.

eBird Data: This allows you to display citizen science data collected via eBird . eBird is not affiliated with the National Park Service. Data from the District of Columbia, Maryland, Virginia and West Virginia is displayed.

Days of data: When you display eBird data you can chose how many days worth of data you wish to see. By default you will see data from the last two weeks, but you can see as little as the previous day's data and as much as the previous month's data.


Map Layers

This control allows you to add an additional data layer onto the map. Currently two options are avaialable:

Ecoregions: This selection will map the Level III ecoregions (Omernik 1995) onto the parks. Ecoregions are areas have similar ecosystems.

Forested Areas: This selection will highlight which areas of the parks are forested. This map is based on information from the 2011 National Land Cover Database (Jin et al. 2013).


Zoom

Zoom to park: You can select a park from the dropdown menu and click on "Go" to zoom the map to that park.

Enlarge plots: By default, the monitoring plots are dispalyed to scale when the map is zoomed in. The enlarge plots control allow you to make then appear 5, 10, or 25 times larger, which is easier to see.


Show

The show panel allows you to hide controls and legends to better view the map. Note that the "Layer Legend" panel will only appear if a map layer is selected.
